    JSON Editor

    Supported Actions

    • Full import: paste `{ "clients": [...], "resources": [...] }`.
    • Update progress: `{ "action": "updateClientProgress", "clientId": "...", "status": "...", "clientType": "TMP", "nextFollowUpDate": "YYYY-MM-DD" }`.
    • Trigger VR&E retention: `{ "action": "updateClientProgress", "clientId": "...", "clientType": "VR&E", "status": "Employed", "employmentDate": "YYYY-MM-DD" }`.
    • Archive closed case: `{ "action": "updateClientProgress", "clientId": "...", "status": "Closed", "archived": true }`.
    • Add contact: `{ "action": "addContactEvent", "clientId": "...", "scenario": "Successful Contact" }`.
    • Generate report: `{ "action": "generateReport", "clientId": "...", "reportType": "Progress Summary" }`.
